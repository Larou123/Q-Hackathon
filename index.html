Triage logic — confidence model

System recommends autonomously. Human can override at any time.

Human review

One or more evidence sources incomplete. Auditor flagged uncertainty. Constraint cannot be confirmed automatically. Confidence 60–84%.

System routes to procurement team for validation.

Reject

System blocks option and logs rejection reason.
