Triage logic — confidence model

Auto-recommend

All evidence sources verified by Auditor. No BOM constraint conflicts. Compliance confirmed. Confidence ≥ 85%.

System recommends autonomously. Human can override at any time.

Human review

One or more evidence sources incomplete. Auditor flagged uncertainty. Constraint cannot be confirmed automatically. Confidence 60–84%.

System routes to procurement team for validation.

Reject

Hard constraint violation confirmed by Auditor. Missing mandatory certification. BOM incompatibility detected. Option blocked regardless of price.

System blocks option and logs rejection reason.

model_training

Learning loop — improves over time

When procurement teams approve, reject, or override Agnes recommendations, those decisions are stored as sourcing intelligence. Human review outcomes on yellow-triage cases refine the confidence model — improving evidence quality scoring, compliance threshold calibration, and BOM constraint matching for future raw materials.
